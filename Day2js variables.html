1. Write a short explanation of what a variable is and why its used in javascript. 

A variable in JavaScript is a named storage location that holds a value, such as a number, string, or object. It allows you to store, retrieve, and manipulate data throughout your code.

        Variables are used to:
        
        1. Store data for reuse without hardcoding values multiple times.
        2. Make code dynamic by changing values based on conditions or user input.
        3. Improve readability by giving meaningful names to data.
        4. Simplify maintenance, since you can update a value in one place rather than throughout the code.
        5. In JavaScript, variables are declared using var, let, or const.

2. Create a simple JavaScript script that declares three different types of variables (var, let, and const) and prints their values to the console. 

        var a=20;
        let b=30;
        const c=40;

        console.log(a);
        console.log(b);
        console.log(c);

        Output:
        20
        30
        40
